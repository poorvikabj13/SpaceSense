# SpaceSense - Video Demo Files

Due to file size limitations, video files are not included in this repository.

## To run the demo:

1. Add your own video files to this directory
2. Name them as:
   - `demo.mp4` (Pedestrian Video 1)
   - `demo2.mp4` (Pedestrian Video 2)
   - `demo3.mp4` (Pedestrian Video 3)

Or update the `VIDEO_MAP` in `app.py` to point to your video files.

## Recommended Video Sources:
- Pedestrian crossing footage
- Crowd monitoring videos
- Public space surveillance footage

**Note:** Ensure videos show people for anomaly detection to work properly.
